  1. 安装主题。

    对于orange：cd到博客根目录，使用git bash或终端输入：

    oranges主题仓库：https://github.com/zchengsite/hexo-theme-oranges

  2. 使用主题（对于oranges主题）：

    在Hexo博客根目录下找到_config.yml文件，修改其中theme字段为主题名oranges

    对于orange的主题配置，需要复制主题文件夹下的_config.yml文件到博客根目录并改名为_config.oranges.yml，后续修改主题操作均在_config.oranges.yml中进行。

    更多主题配置请参考oranges主题仓库：https://github.com/zchengsite/hexo-theme-oranges
