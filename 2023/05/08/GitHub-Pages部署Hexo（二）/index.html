1 为Hexo博客配置主题

1.1 初始配置主题

  1. 安装主题。

    对于orange：cd到博客根目录，使用git bash或终端输入：

    oranges主题仓库：https://github.com/zchengsite/hexo-theme-oranges

  2. 使用主题（对于oranges主题）：

    在Hexo博客根目录下找到_config.yml文件，修改其中theme字段为主题名oranges

    对于orange的主题配置，需要复制主题文件夹下的_config.yml文件到博客根目录并改名为_config.oranges.yml，后续修改主题操作均在_config.oranges.yml中进行。

2 开始用Hexo写作

你可以执行下列命令来创建一篇新文章或者新的页面。

您可以在命令中指定文章的布局（layout），默认为 post，可以通过修改 _config.yml 中的 default_layout 参数来指定默认布局。

布局（Layout）

Hexo 有三种默认布局：post、page 和 draft。在创建这三种不同类型的文件时，它们将会被保存到不同的路径；而您自定义的其他布局和 post 相同，都将储存到 source/_posts 文件夹。

布局 路径
post source/_posts
page source
draft source/_drafts
